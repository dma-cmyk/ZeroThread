# Release Notes - ZeroThread

## [1.3.0] - 2026-03-20
### 追加
- **緊急アプリ初期化機能**: サイドパネル最下部とプロフィールページに、ブラウザの全データを一括削除する機能を追加。
- **2段階認証UI**: 初期化ボタンに確認ステップを追加し、誤操作を防止。
- **デバッグツール**: プロフィールページに Local Storage ビューアと秘密鍵パーサーを追加。

### 修正
- **コミュニティ削除の整合性**: コミュニティ削除時に、関連する投稿・コメント・メンバーシップも localStorage から正しく削除されるよう修正。
- **ゴースト投稿のフィルタリング**: すでに削除されたコミュニティに紐付く投稿がフィードに表示されないよう、フロントエンド側で自動フィルタリングを強化。
- **リアルタイム更新**: コミュニティの作成・削除がサイドバーに即座に反映されるよう改善。

### システム
- **Firebase初期化**: Firestore上の全データをクリーンアップ（開発用）。

---

## [1.2.0] - 2026-03-18
### 追加
- **モデレーション機能**: 投稿の非表示、ユーザーの追放管理機能。
- **匿名モード**: 匿名での投稿・コメント投稿機能。
- **マイクサポート**: 音声入力に関連する基盤の導入。
- **プライバシー設定**: 設定画面の簡略化とUI改善。

---

## [1.1.0] - 2026-03-17
### 追加
- **暗号化基盤**: Web Crypto API を利用したアカウント秘密鍵の生成と管理。
- **デモモード**: localStorage を利用したバックエンドなしでの動作モード。
- **認証システム**: 秘密鍵ベースのログイン/新規登録フロー。

---

## [1.0.0] - 2026-03-17
### 追加
- **プロジェクト初期化**: React + Vite + TypeScript によるベース構築。
- **基本UI**: ホーム、人気、新着フィードのレイアウト実装。
- **Firebase連携**: Firestore との基本的な接続。
